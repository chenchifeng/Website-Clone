# 介绍

## 主要概述
1. 这是一个Web网页制作的期末作品，主要是仿制一个网站。

2. 这个仿制出来的网站，要有后台，有数据库，可以在后台实现信息的增删查改。这其实就可以差不多把一个网站当成了一个通用的模板，然后根据需求来更改信息来变成另一个网站了，也可以发布修改内容。

3. 使用的是后台模板 + Access数据库 + Asp。

## 过程
0. 选择需要仿制的合适的网站

1. 收集保存需要仿制的网站的资源,包括html,css,js,还有其他图片资源等。（注意：有些例如background图片文件这种保存不了的，可以直接用迅雷之类的下载器，或者一些浏览器自带媒体资源下载插件下载）

2. 整理好相关页面资源，放到相应的目录下。

3. 对页面的修正，链接资源文件。

4. 对html页面，相同部分进行整合，用include header footer等。这里用到的后端语言是Asp。

5. 创建数据库以及链接后台。这里用到的数据库是Access，后台是尺度后台。

6. 对页面需要链接数据库的地方进行处理，把原来的html文本内容，替换成使用asp链接数据库的内容。
